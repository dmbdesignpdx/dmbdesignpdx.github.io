Flowarchy

A CSS Structure Flow for Designers

Flowarchy is a way of structuring the flow of CSS properties of an element through a hierarchy. The goal is to help organize your CSS and make finding CSS properties faster in elements with lengthy properties. This CSS flow was produced from a designer's point of view.
